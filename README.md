# Personal Portfolio Website

## Description
This is a personal portfolio website showcasing my projects, skills, and experience. Built with modern web technologies to highlight my work and provide an easy way to connect.

## Features
- Interactive project demos
- Responsive design that works on all devices
- Easy navigation
- Contact form to reach out

## Technologies Used
- HTML (71.2%)
- JavaScript (28.8%)

## Setup Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/warda565/warda565.github.io
   ```
2. Navigate to the project directory:
   ```bash
   cd warda565.github.io
   ```
3. Open `index.html` in your browser to view the portfolio.
4. For local development, you can use a simple HTTP server:
   ```bash
   npx http-server .
   ```
